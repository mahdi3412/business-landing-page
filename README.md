# Modern Business Landing Page

A responsive landing page built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- Hero section with CTA
- Services grid
- About section
- Testimonials
- Contact form with Formspree
- Fully responsive design
- Deployed on Vercel

## Tech Stack

- Next.js
- Tailwind CSS
- Framer Motion
- Formspree

## Getting Started

```bash
npm install
npm run dev
```
