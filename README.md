# Modern Business Landing Page

A responsive and animated landing page for a fictional business, built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- Clean and modern UI
- Fully responsive
- Smooth animations
- Contact form with Formspree
- Deployed on Vercel
